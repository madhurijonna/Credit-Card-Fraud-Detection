# Credit-Card-Fraud-Detection
## Project Overview
This project detects fraudulent credit card transactions using Machine Learning models.
## Models Used
- Logistic Regression
- Random Forest
- XGBoost
## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn
## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Handling Imbalanced Data
4. Model Building
5. Model Evaluation
## Results
- Logistic Regression Accuracy: 96%
- Random Forest Accuracy: 99%
- XGBoost Accuracy: 99%
